## Usage

- by-name

```bash
python main.py by-name "A Beginners Guide to Python 3 Programming"
```

- by-xlsx

> Note: default file is *Free+English+textbooks.xlsx*

```bash
python main.py by-xlsx

Or

python main.py by-xlsx FILENAME.xlsx
```
